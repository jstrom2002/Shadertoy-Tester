# Shadertoy-Tester #
OpenGL program to quickly load and test Shadertoy's shaders locally.

### Dependencies ###
OpenGL  
glew  
glfw3  
STBI  

### Notes ###
Textures and other media taken from: https://shadertoyunofficial.wordpress.com/2019/07/23/shadertoy-media-files/.
